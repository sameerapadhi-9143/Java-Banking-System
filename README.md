# Java- Banking Management System

A simple console-based Banking Management System built using **Java, JDBC, and MySQL**.  
This project simulates basic banking operations and demonstrates how Java applications interact with a relational database.

---

## 🚀 Features

- User Registration and Login  
- Deposit and Withdraw Operations  
- Balance Inquiry  
- Transaction History Tracking  

---

## 🛠️ Tech Stack

- Java (Core Java)
- JDBC (Java Database Connectivity)
- MySQL

---

##  Project Structure

src/main/java/com/example/testjdbc/
- Main.java → Application entry point  
- UserService.java → Business logic  
- DBConnection.java → Database connectivity  

---

## ⚙️ Setup Instructions

1. Create a MySQL database named `bank_system`  
2. Create required tables (`users`, `accounts`, `transactions`)  
3. Update database credentials in `DBConnection.java`  
4. Run `Main.java` to start the application  

---

##  Purpose

This project was developed to understand:
- JDBC connectivity  
- Database operations (CRUD)  
- Structuring a backend application in Java
